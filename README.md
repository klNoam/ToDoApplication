ToDo List Application:

A simple ToDo list web application running on local host. Users can add, complete, and delete tasks. 

Setup Instructions:
--> Clone the Repository
(Terminal) git clone https://github.com/klNoam/ToDoApplication
--> Install dependencies (node package manager)
(Terminal) npm install
--> Run the backend server
(Terminal) node server.js
--> Open the frontend
(Browser) http://localhost:3000


Notes:

--> Features of the app
 - Type in task in the input field and click the "Add Task" button to add it to the ToDo list
 - Press the checkbox next to a task in order to complete the task
 - Press the delete button next to a task in order to get rid of the task from the list

--> Bugs of the app
- Empty tasks are able to be added
- Tasks are stored in memeory, meaning that restarting the server clears the tasks
